### Description

This web application manages the upload of video clips around gaming themes. It uses cloud storage through CloudStorage, Firebase authentication, and Firestore as the database.

### Features

#### Home

Homepage displaying the latest videos uploaded by different users. For each video, the user who uploaded it, the date, and a video thumbnail are shown.

![Clips Home](/images/projects/home-clips.webp)

![Clips Home 2](/images/projects/home-clips-2.webp)

#### Registration and Authentication

Full use of the application requires the user to be authenticated, which is obtained through registration.

![Clips Registration](/images/projects/register-clips.webp)

#### Video Clip Management

In this module you can edit the video clip title, as well as delete it or upload more videos.

![Clips Management](/images/projects/manage-clips.webp)

#### Video Clip Playback

In this module you can view videos in fullscreen.

![Clips Video Play](/images/projects/video-play.webp)
