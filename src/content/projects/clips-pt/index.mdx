### Descrição

Esta aplicação web gerencia o upload de clipes de vídeo sobre temas de videogames. Utiliza armazenamento em nuvem através do CloudStorage, autenticação do Firebase e Firestore como banco de dados.

### Funcionalidades

#### Home

Página inicial que mostra os últimos vídeos carregados pelos diferentes usuários. Para cada vídeo, é mostrado o usuário que carregou o vídeo, a data e uma miniatura do vídeo.

![Home de Clips](/images/projects/home-clips.webp)

![Home de Clips 2](/images/projects/home-clips-2.webp)

#### Registro e Autenticação

Para uso completo da aplicação, é necessário que o usuário esteja autenticado, sendo essa autenticação obtida através de um registro.

![Registro de Clips](/images/projects/register-clips.webp)

#### Gerenciamento de Videoclipes

Neste módulo é possível editar o título do videoclipe, bem como excluí-lo ou carregar mais vídeos.

![Gestão de Clips](/images/projects/manage-clips.webp)

#### Reprodução de Videoclipes

Neste módulo é possível visualizar os vídeos em tela cheia.

![Reprodução de vídeo de Clips](/images/projects/video-play.webp)
