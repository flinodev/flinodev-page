### Descrição

Este sistema web possui todas as características de um ecommerce tradicional, desde a visualização do catálogo completo de produtos, carrinho de compras, pagamentos online, administração de catálogo de produtos e usuários, autenticação, proteção de rotas e mais. Desenvolvido com Next como framework full-stack, Postgres como banco de dados, JWT e Tailwind.

### Funcionalidades

#### Home

Nesta tela é mostrada uma paginação dos produtos disponíveis na loja virtual. Na barra superior, ao centro é possível filtrar o conteúdo por categorias. Embora o sistema possua autenticação, esta página pode ser visualizada de forma não autenticada.

![Home da teslo-shop](/images/projects/home-teslo.webp)

#### Detalhe do Produto

Para cada produto é mostrada a informação mais relevante, como seja, seu preço, a quantidade de produtos em estoque, os tamanhos disponíveis e uma descrição do mesmo. Além da funcionalidade de adicioná-lo ao carrinho de compras.

![Detalhe de produto na teslo-shop](/images/projects/product-teslo.webp)

#### Carrinho de Compras

Depois de adicionar alguns produtos, estes vão sendo armazenados no carrinho de compras para sua posterior aquisição. O armazenamento é feito através do localStorage do navegador. Portanto, para seu uso, não é necessário estar autenticado.

![Carrinho de compras na teslo-shop](/images/projects/cart-teslo.webp)

#### Login

Para realizar o pagamento dos produtos, e realizar um conjunto de operações além da exploração do catálogo de produtos, é necessário autenticar-se.

![Autenticação na teslo-shop](/images/projects/auth-teslo.webp)

#### Checkout

Para confirmar o pedido é necessário adicionar o endereço de entrega dos produtos, sendo que temos a opção de salvá-lo no banco de dados (Lembrar endereço), ou não, caso em que só será armazenado no localStorage do navegador.

![Checkout na teslo-shop](/images/projects/checkout-teslo.webp)

Posterior à verificação do nosso pedido, este é armazenado no banco de dados e está pronto para ser pago. A confirmação do nosso pedido impacta no estoque dos nossos produtos, de tal maneira que outros usuários ao solicitar o conteúdo dos nossos produtos possam ver a disponibilidade dos produtos até esse momento.

![Checkout 2 na teslo-shop](/images/projects/checkout-2-teslo.webp)

#### Pagamentos Online

O sistema de comércio eletrônico admite pagamentos online através do Paypal assim como cartão de débito e crédito.

![Pagamentos online na teslo-shop](/images/projects/paypal-teslo.webp)

#### Administração de Catálogo de Produtos

Esta página, protegida e permitida apenas para usuários administradores, permite dar manutenção ao catálogo de produtos. Tanto editar os já existentes, como adicionar novos produtos.

![Administração de produtos teslo-shop](/images/projects/admin-products-teslo.webp)

Para cada produto podemos modificar qualquer campo de informação incluindo as fotografias. Estas mudanças serão persistentes no banco de dados e se refletirão para todos aqueles usuários que visitarem o site.

![Edição de produtos na teslo-shop](/images/projects/edit-product-teslo.webp)
