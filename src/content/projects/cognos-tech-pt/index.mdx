### Descrição

Plataforma web como ferramenta de apoio no ensino de cursos de inglês e simulação de provas de admissão para o ensino médio. Desenvolvida com Angular 14, Angular Material, Firebase Auth, Firestore como banco de dados e Cloud Functions com NodeJS.

### Funcionalidades

#### Dashboard

A ferramenta possui um dashboard que entre outras coisas mostra:

- Progresso do aluno
- Tempo dedicado na plataforma
- Cursos em que está inscrito
- Cursos disponíveis na plataforma

Dentro de cada curso é possível consultar uma breve descrição do mesmo, o conteúdo dividido por módulos, o tempo dedicado a cada um, bem como o progresso atual.

![Dashboard do CognosTech](/images/projects/cognos-main.webp)

#### Módulos por Matéria

Para cada módulo há uma série de lições, exercícios e recursos adicionais (pdf, fotos, áudios, apresentações, etc.) para cada tema tratado no módulo.

![Resumo do módulo no CognosTech](/images/projects/module-resume.webp)

#### Exercícios por Lição

Na seção de exercícios podemos encontrar questões de múltiplos tipos:

- Múltipla escolha
- Drag and Drop
- Entrada de texto
- Saída de áudio

Cada questão tem um peso em pontos, que serão acumulados ou não, de acordo com o que o usuário responder. Além disso, para cada questão é registrado o tempo utilizado pelo usuário para respondê-la.

![Tipos de exercícios no CognosTech](/images/projects/exercises.webp)

#### Características Adicionais

A aplicação possui outros módulos como o resumo das respostas antes do envio, um módulo para os dados do usuário bem como sua edição, ranking semanal e global, um módulo para o acompanhamento dos pagamentos dos usuários, entre outros.

![Outros módulos no CognosTech](/images/projects/others.webp)

Adicionalmente está sendo trabalhada a otimização dos módulos para a administração do conteúdo, como seja, o upload de lições, exercícios e conteúdo adicional.
