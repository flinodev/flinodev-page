### Description

Web platform as a support tool for teaching English courses and simulating entrance exams for high school level. Developed with Angular 14, Angular Material, Firebase Auth, Firestore as database, and Cloud Functions with NodeJS.

### Features

#### Dashboard

The tool has a dashboard that among other things shows:

- Student progress
- Time spent on the platform
- Courses enrolled in
- Available courses on the platform

Within each course you can consult a brief description, content sectioned by modules, time spent on each, as well as current progress.

![CognosTech Dashboard](/images/projects/cognos-main.webp)

#### Subject Modules

Each module has a series of lessons, exercises, and additional resources (pdf, photos, audios, presentations, etc.) for each topic covered in the module.

![Module summary in CognosTech](/images/projects/module-resume.webp)

#### Lesson Exercises

In the exercises section we can find multiple types of questions:

- Multiple choice
- Drag and Drop
- Text input
- Audio output

Each question has a point weight, which will accumulate or not, according to what the user answers. Additionally, for each question the time spent by the user to answer it is recorded.

![Exercise types in CognosTech](/images/projects/exercises.webp)

#### Additional Features

The application has other modules such as the summary of answers prior to submitting responses, a module for user data as well as its edition, weekly and global ranking, a module for tracking user payments, among others.

![Other modules in CognosTech](/images/projects/others.webp)

Additionally, work is being done on optimizing modules for content administration, such as uploading lessons, exercises, and additional content.
