### Description

This mobile application allows the management of an ecommerce inventory. It enables changing the characteristics of each product, as well as adding new items to the catalog.

### Features

#### Login

Access and product management require the user to be authenticated. For this, a login form is provided where the user enters their credentials. If correct, the application will grant access to the inventory.

![Teslo-shop login](/images/projects/teslo_app_1.webp)

#### Home

On this first screen, all available products in the virtual store are displayed with their respective name and a product image. In the lower right corner, you can see a button to add one more product to the inventory.

![Teslo-shop home](/images/projects/teslo_app_3.webp)

#### Add New Product

On this screen you can add a new product, where required fields have their respective validation. The photo can also be uploaded via the phone's camera or gallery.

![New product in teslo-shop](/images/projects/teslo_app_6.webp)

#### Product Details

On this screen you can modify a product, where fields have their respective validation. Here it is possible to edit (remove, add, etc.) the photos corresponding to the product in question.

![Product details in teslo-shop](/images/projects/teslo_app_4.webp)

All operations are persistent. Behind the application there is a backend that handles client requests, which in this case is our app. Changes are saved in a Postgres database and photos in a third-party provider.

The code for this application was developed implementing SOLID principles and Clean Architecture.
